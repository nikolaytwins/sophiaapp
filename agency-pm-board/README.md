# Канбан проектов (ПМ)

Отдельный сервис для проджект-менеджера: только карточки проектов, дедлайны и статусы. Без сумм и оплат.

- **Страница канбана:** после деплоя — `http://178.72.168.156/pm-board` (или корень, если открывать на порту 3003).
- При добавлении проекта на http://178.72.168.156/agency в Twinworks автоматически создаётся карточка здесь со статусом «не начато».

## Статусы

не начато, написание текстов, согласование текстов, дизайн первого экрана/слайда, согласование первого экрана/слайда, дизайн, согласование дизайна, верстка, согласование верстки, готов, пауза.

## API (для будущего бота)

- `GET /pm-board/api/cards` — список карточек (id, name, deadline, status, source_project_id, created_at, updated_at).
- `PATCH /pm-board/api/cards/:id` — обновить карточку (body: { status?, name?, deadline? }).

Сумм и полей оплаты в API нет.

## Запуск

- Порт: 3003.
- БД: SQLite `data/pm-board.db` в корне проекта.
- Деплой: `deploy/agency-pm-board.service`, nginx — `twinworks/deploy/nginx-twinworks.conf` (location /pm-board).

## Twinworks

В Twinworks в `.env` или переменных окружения можно задать `PM_BOARD_URL=http://127.0.0.1:3003` (по умолчанию так и есть). Нужен для вызова POST /api/cards при создании проекта.
