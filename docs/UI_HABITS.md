# UI: ритм (привычки и день)

Экраны **Привычки** (`/sophia/habits`) и **День** (`/sophia/day`) используют один визуальный язык: тёмный холст `#030304`, градиент холста как в `HabitsScreen`, фиолетовый акцент, те же карточки и типографика шапки («Ритм» + заголовок + дата).

Данные **дневника состояния** (утро / вечер / заметка) хранятся в `useDayJournalStore`, экспорт — JSON через `buildDayJournalExportDoc()` (`schema: sophia.dayJournal.v1`).

## База

- Фон страницы: тёмный (`#030304`), лёгкий градиент по холсту (как в `HabitsScreen`).
- Акцент: фиолетовая палитра (`src/theme/`), не отдельные «золотые» или иные декоративные рамки для блоков того же уровня, что и карточки привычек.
- Карточки: паттерн `SurfaceCard` — фон `rgba(18,18,22,0.92)`, обводка `rgba(255,255,255,0.07)` или с лёгким glow `rgba(168,85,247,0.22)`; при необходимости сверху слабый градиент `rgba(168,85,247,0.07) → transparent`.
- Подписи секций: `caption`, верхний регистр, приглушённый белый (`~0.32` opacity), `letterSpacing` ~1.3 (как у блоков **МЕСЯЦ** внутри карточек).

## Блок «выполнение за месяц» (агрегат по всем привычкам)

Должен визуально совпадать с карточками: тот же `SurfaceCard` + glow и тот же фиолетовый градиент-оверлей, что и у пустого состояния / списка — **без** отдельной золотистой рамки и янтарного градиента.

Кольцо прогресса (`ProgressRing`) остаётся носителем процента; трек и дуга берутся из темы (`colors.borderStrong`, `colors.accent`).

## Токены

Детали цветов и типографики — в `src/theme/`; при новых экранах привычек наследуйте паттерны с этого экрана, а не выдумывайте параллельную «премиальную» палитру для тех же компонентов.
