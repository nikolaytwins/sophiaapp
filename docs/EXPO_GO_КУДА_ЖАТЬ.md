# Куда жать (совсем по шагам)

## Где живёт приложение

Код и Metro должны крутиться **на сервере** (тот же компьютер/VPS, где лежит `sophia-os`).  
Телефон только **подключается** к уже запущенному Metro.

---

## А) На сервере (через SSH в терминал)

Скопируй **три строки** по очереди, каждая Enter в конце:

```bash
cd /root/.openclaw/workspace/sophia-os
npm install
npm run start:lan
```

После третьей команды **ничего не закрывай** — в терминале должен идти лог Metro.  
Найди строку **`Packager host для QR:`** — там IP (например `178.72.168.156`). Этот IP понадобится ниже.

Остановить Metro: в том же терминале **Ctrl+C**.

---

## Б) На iPhone — сначала Safari (проверка сети)

1. Открой приложение **Safari** (иконка компаса).
2. Вверху ткни в **адресную строку**.
3. **Сотри** всё, что там есть.
4. **Вставь** (замени IP на свой из лога, если другой):

   `http://178.72.168.156:8081`

5. Нажми **Перейти** / **Go**.

- Если **что-то открылось** (хоть страница Expo) — до сервера **дошло**. Иди к пункту В.
- Если **крутится и ошибка** — фаервол или Metro не запущен на сервере.

---

## В) На iPhone — Expo Go

1. Открой приложение **Expo Go** (иконка чёрная с белым треугольником).
2. На экране найди что-то вроде **Enter URL manually** / **Ввести URL вручную** — **нажми**.
3. В поле ввода вставь (**без** `http`, именно `exp://`):

   `exp://178.72.168.156:8081`

   (снова подставь **свой** IP из лога сервера)

4. Нажми **Connect** / **Подключить**.

---

## Если «не понимаю, что жать»

Минимум:

| Где        | Что сделать                                      |
|-----------|---------------------------------------------------|
| **Сервер** | SSH → три команды из блока А → окно не закрывать   |
| **Safari** | Адресная строка → `http://ТВОЙ_IP:8081`          |
| **Expo Go**| Ввод URL вручную → `exp://ТВОЙ_IP:8081`           |

Автопроверка **на сервере** (после того как Metro запустишь):

```bash
cd /root/.openclaw/workspace/sophia-os
npm run check:mobile
```

Должно быть: порт **8081** слушает **0.0.0.0**, не пусто.
