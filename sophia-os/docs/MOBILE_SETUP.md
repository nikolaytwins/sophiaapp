# Как открыть Sophia OS на iPhone (сервер на Selectel)

Короткий сценарий **Expo Go**: сначала [EXPO_GO.md](./EXPO_GO.md). Этот файл — детали для **VPS + порт 8081** и «кто что делает».

## Кто что делает

### Ты (только у тебя есть доступ)

1. **Фаервол Selectel** — правило **входящий TCP 8081** на IP твоего сервера. Без этого iPhone из интернета **не достучится** до Metro.  
   **Подробная пошаговая инструкция:** [SELECTEL_FIREWALL_8081.md](./SELECTEL_FIREWALL_8081.md).
2. **Фаервол на самой ВМ** (если включён `ufw` / iptables) — разрешить **8081/tcp** входящий.
3. **Запуск Metro** на сервере командой ниже и **оставить терминал открытым**, пока тестируешь с телефона.
4. **Expo Go** — вручную ввести `exp://ТВОЙ_IP:8081` (IP из строки `Packager host` в логе).
5. **Проверка с телефона** — в Safari открыть `http://ТВОЙ_IP:8081`. Если страница не грузится, проблема сеть/фаервол, не код приложения.

### Я (агент в репозитории)

- Уже настроены скрипты `start:lan`, `start:tunnel`, документация.
- Могу **дополнять доки и скрипты** (как этот файл и `scripts/check-mobile-connection.sh`).
- **Не могу**: зайти в панель Selectel, открыть порты на твоём сервере, запустить долгоживущий Metro на твоей машине за тебя или проверить твой реальный фаервол.

## Быстрая самопроверка на сервере (SSH)

```bash
cd /root/.openclaw/workspace/sophia-os
bash scripts/check-mobile-connection.sh
```

Смотри вывод: слушает ли **0.0.0.0:8081**, отвечает ли localhost.

## Что уже сделано в проекте

- `app/index.tsx` — с корня `/` редирект на главный экран (в браузере не будет «Экран не существует»).
- `npm run start:lan` — запуск Metro с **публичным IP** в QR (`REACT_NATIVE_PACKAGER_HOSTNAME`).
- `npm run start:tunnel` — запуск через **tunnel** (ngrok), если LAN не подходит.

## Способ 1: LAN + публичный IP (рекомендуется на VPS)

1. **В панели Selectel** открой для сервера **входящий TCP порт 8081** (и при необходимости **19000**, **19001** — если Expo попросит в логах).
2. На сервере:
   ```bash
   cd /root/.openclaw/workspace/sophia-os
   npm run start:lan
   ```
3. На iPhone: приложение **Expo Go** → **Enter URL manually** (ввод ссылки вручную) и вставь:
   ```text
   exp://178.72.168.156:8081
   ```
   Если IP сервера другой — смотри строку `Packager host для QR` в выводе команды.

4. Телефон должен иметь доступ в интернет к этому IP (обычно так и есть).

## Способ 2: Tunnel (если ngrok работает)

```bash
cd /root/.openclaw/workspace/sophia-os
npm run start:tunnel
```

Иногда нужно: `npx expo login`. Если видишь ошибку Ngrok — смотри [status.ngrok.com](https://status.ngrok.com/) или используй способ 1.

## Ошибка про libnspr4 (DevTools)

Это предупреждение на сервере; Metro обычно всё равно работает. Чтобы убрать, когда `apt` свободен:

```bash
sudo apt-get update && sudo apt-get install -y libnss3 libnspr4
```

## Веб с ПК

SSH-туннель: `ssh -L 8081:127.0.0.1:8081 root@ТВОЙ_IP` → в браузере `http://localhost:8081`.
